# restful-courses

## Description:

This project works as a practice to start using NodeJs and its components following the tutorial: [Build RESTful APIs with Node and Express by Mosh](https://www.youtube.com/watch?v=pKd0Rpw7O48)

## Technologies:

- NodeJs
- Express module
- Joi module
- Postman
